ホームディレクトリにdotfilesをクローン

```sh
git clone
```

シンボリックリンク作成

```sh
ln -sv ~/dotfiles/.zshrc ~/
```

必要に応じて環境変数を追加する

```sh
vim ~/dotfiles/zsh/env.zsh
```
